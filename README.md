This is the source code for the [arduino wiki website](https://wiki.arduinodiscord.cc) documentation site. It is built using [VitePress](https://vitepress.vuejs.org/).

## Development

### Install dependencies

```bash
cd vitepress && yarn
```

### Start dev server

```bash
yarn dev
```

### Build

```bash
yarn build
```

### Serve

```bash
yarn serve
```
